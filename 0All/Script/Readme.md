## BCHjar.py

BCH是用来求放入多个文件的最适宜lambda参数，lambdaTestResult为多个文件一起计算的结果。无需修改config.json的参数。

将聚类好的csv文件放到Cluster文件夹下

result即为最优凹壳结果
